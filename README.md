## **Using üì¶**

1. Clone Template

```
git clone https://github.com/ItsHuyne/user-manager-reactjs
```

2. Open 2 terminal

First terminal

```
cd frontend
```

Second terminal:

```
cd backend
```

3. Install Packages (both frontend and backend)

```
yarn install
```

4. Start Project

Frontend:

```
yarn dev
```

Backend: 

```
npm start (yarn start)
```
